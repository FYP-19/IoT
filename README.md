## Essential Linux Commands 

| Function                | Linux Command                                |
|-------------------------|----------------------------------------------|
| Check camera status    | vcgencmd get_camera            |
| Real time camera feed   | raspivid -t 0            |
| Capture an image        |  raspistill -o <image_name>.jpg |
| Watch the image        |  eog <image_name>.jpg |
